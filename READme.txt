Afin de faire fonctionner le programme correctemment, il suffit de d'appuyer sur CTRL+E.

le menu va demander le champignon que vous voulez comparer, il faudra  écrire l'indice (dans le tableau excel)  du champignon que l'on veut determiner 
il va ensuite demander le nombre de voisins : il faudra alors rentrer un nombre dans la console

Pour pouvoir tester des nouveaux champignon, il suffit de rentrer les valeurs à la suite du fichier excel fournit de base (5000 champigons)

le programme affichera alors les voisins les plus proche, la distance avec le champignon que l'on souhaites determiner, 
l'indice des voisins dans le tableau et indiquera si le champignons testé est mangeable ou non.



pour le test final des 10 champignons, j'ai copié collé les attributs a la suite de la liste des 5000 attributs afin de pouvoir les comparé aux autres.
